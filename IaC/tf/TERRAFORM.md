## Install
```bash
# add brew package
brew tap hashicorp/tap

# install tf
brew install hashicorp/tap/terraform

# check install
terraform -version
Terraform v1.4.2
on darwin_arm64
+ provider registry.terraform.io/navercloudplatform/ncloud v2.3.10
```


## Set alias
```bash
alias tf=terraform
```
